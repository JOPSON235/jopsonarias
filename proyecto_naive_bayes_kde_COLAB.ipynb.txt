    "def balance_dataset(X_train, y_train, random_state=42, ratio=3):\n",
    "    \"\"\"\n",
    "    Submuestreo aleatorio de la clase mayoritaria.\n",
    "    Parámetros:\n",
    "        ratio: cuántas muestras de la clase 0 por cada muestra de la clase 1.\n",
    "    \"\"\"\n",
    "    X_df = pd.DataFrame(X_train)\n",
    "    X_df['target'] = y_train\n",
    "\n",
    "    majority = X_df[X_df['target'] == 0]\n",
    "    minority = X_df[X_df['target'] == 1]\n",
    "\n",
    "    n_minority = len(minority)\n",
    "    n_majority_target = min(len(majority), n_minority * ratio)\n",
    "\n",
    "    majority_down = resample(majority, replace=False,\n",
    "                             n_samples=n_majority_target,\n",
    "                             random_state=random_state)\n",
    "\n",
    "    df_balanced = pd.concat([majority_down, minority]) \\\n",
    "                    .sample(frac=1, random_state=random_state) \\\n",
    "                    .reset_index(drop=True)\n",
    "\n",
    "    X_bal = df_balanced.drop('target', axis=1).values\n",
    "    y_bal = df_balanced['target'].values\n",
    "    return X_bal, y_bal\n",
    "\n",

    "## 🧠 4. Implementación del Clasificador Naive Bayes con KDE\n",
    "\n",
    "Se implementa desde cero la clase `KDENaiveBayes`, compatible con la API de scikit-learn.\n",
    "\n",
    "**Fundamento:** En lugar de asumir $P(x_i | y) = \\mathcal{N}(x_i | \\mu, \\sigma^2)$, se estima la densidad de forma no paramétrica:\n",
    "\n",
    "$$\\hat{f}(x) = \\frac{1}{nh} \\sum_{i=1}^{n} K\\left(\\frac{x - x_i}{h}\\right)$$\n",
    "\n",
    "La predicción usa log-probabilidades para estabilidad numérica:\n",
    "\n",
    "$$\\hat{y} = \\arg\\max_y \\left[ \\log P(y) + \\sum_{j=1}^{d} \\log \\hat{f}_j(x_j | y) \\right]$$"
   ]
  },
  {
   "cell_type": "code",
   "metadata": {
    "id": "kde_class"
   },
   "source": [
    "class KDENaiveBayes(BaseEstimator, ClassifierMixin):\n",
    "    \"\"\"\n",
    "    Clasificador Naive Bayes con verosimilitud estimada por KDE.\n",
    "\n",
    "    Parámetros\n",
    "    ----------\n",
    "    kde_type : {'gaussian', 'parzen', 'silverman'}\n",
    "        - 'gaussian'  : Kernel gaussiano, bandwidth definido por el usuario.\n",
    "        - 'parzen'    : Kernel rectangular (tophat), bandwidth fijo.\n",
    "        - 'silverman' : Kernel gaussiano con h = 1.06 * sigma * n^(-1/5) por feature.\n",
    "    bandwidth : float\n",
    "        Ancho de banda para 'gaussian' y 'parzen'. Ignorado en 'silverman'.\n",
    "    \"\"\"\n",
    "\n",
    "    def __init__(self, kde_type='gaussian', bandwidth=1.0):\n",
    "        self.kde_type = kde_type\n",
    "        self.bandwidth = bandwidth\n",
    "\n",
    "    def _silverman_bandwidth(self, data):\n",
    "        \"\"\"Regla de Silverman: h = 1.06 * sigma * n^(-1/5)\"\"\"\n",
    "        sigma = np.std(data, ddof=1)\n",
    "        sigma = max(sigma, 1e-8)        # Evitar BW = 0 para variables constantes\n",
    "        h = 1.06 * sigma * (len(data) ** (-1/5))\n",
    "        return max(h, 1e-4)\n",
    "\n",
    "    def fit(self, X, y):\n",
    "        \"\"\"\n",
    "        Estima P(y) y los KDEs P(x_j | y) para cada clase y feature.\n",
    "        \"\"\"\n",
    "        self.classes_ = np.unique(y)\n",
    "        self.n_features_ = X.shape[1]\n",
    "\n",
    "        # Probabilidades a priori P(y)\n",
    "        self.priors_ = {c: np.mean(y == c) for c in self.classes_}\n",
    "\n",
    "        # KDE por clase y por feature\n",
    "        self.kdes_ = {}\n",
    "        for c in self.classes_:\n",
    "            X_c = X[y == c]\n",
    "            self.kdes_[c] = []\n",
    "            for j in range(self.n_features_):\n",
    "                data_j = X_c[:, j]\n",
    "\n",
    "                if self.kde_type == 'silverman':\n",
    "                    h = self._silverman_bandwidth(data_j)\n",
    "                    kde = KernelDensity(kernel='gaussian', bandwidth=h)\n",
    "                elif self.kde_type == 'gaussian':\n",
    "                    kde = KernelDensity(kernel='gaussian', bandwidth=self.bandwidth)\n",
    "                elif self.kde_type == 'parzen':\n",
    "                    kde = KernelDensity(kernel='tophat', bandwidth=self.bandwidth)\n",
    "                else:\n",
    "                    raise ValueError(f'kde_type desconocido: {self.kde_type}')\n",
    "\n",
    "                kde.fit(data_j.reshape(-1, 1))\n",
    "                self.kdes_[c].append(kde)\n",
    "        return self\n",
    "\n",
    "    def predict_proba(self, X):\n",
    "        \"\"\"\n",
    "        Calcula P(y | x) usando log-probabilidades (estabilidad numérica).\n",
    "        \"\"\"\n",
    "        n_samples = X.shape[0]\n",
    "        log_post = np.zeros((n_samples, len(self.classes_)))\n",
    "\n",
    "        for i, c in enumerate(self.classes_):\n",
    "            # Log-prior\n",
    "            log_post[:, i] = np.log(self.priors_[c])\n",
    "            # Sumar log-verosimilitudes de cada feature\n",
    "            for j in range(self.n_features_):\n",
    "                log_lik = self.kdes_[c][j].score_samples(X[:, j].reshape(-1, 1))\n",
    "                log_post[:, i] += log_lik\n",
    "\n",
    "        # Truco del máximo para estabilidad numérica antes de softmax\n",
    "        log_post -= log_post.max(axis=1, keepdims=True)\n",
    "        posteriors = np.exp(log_post)\n",
    "\n",
    "        # Normalizar + limpiar NaN/Inf\n",
    "        row_sums = posteriors.sum(axis=1, keepdims=True)\n",
    "        row_sums = np.where(row_sums == 0, 1e-300, row_sums)\n",
    "        posteriors /= row_sums\n",
    "        posteriors = np.nan_to_num(posteriors, nan=0.5, posinf=1.0, neginf=0.0)\n",
    "        posteriors /= posteriors.sum(axis=1, keepdims=True)\n",
    "\n",
    "        return posteriors\n",
    "\n",
    "    def predict(self, X):\n",
    "        \"\"\"Predice la clase más probable.\"\"\"\n",
    "        return self.classes_[np.argmax(self.predict_proba(X), axis=1)]\n",
    "\n",
    "\n",
    "print('✅ Clase KDENaiveBayes definida.')\n",
    "print(f'   Métodos soportados: gaussian | parzen | silverman')"
   ],

    "# Grid de búsqueda (escala logarítmica alrededor de Silverman)\n",
    "bw_grid = np.round(np.logspace(-2, 0.5, 15), 4)\n",
    "print(f'Grid de bandwidth a evaluar: {bw_grid}')\n",
    "print()\n",
    "\n",
    "# Usar submuestra para acelerar la búsqueda\n",
    "np.random.seed(42)\n",
    "n_sample = 2000\n",
    "idx_sample = np.random.choice(len(X_scaled), n_sample, replace=False)\n",
    "X_sub, y_sub = X_scaled[idx_sample], y[idx_sample]\n",
    "\n",
    "cv_bw = StratifiedKFold(n_splits=5, shuffle=True, random_state=42)\n",
    "bw_scores = []\n",
    "\n",
    "print('Optimizando bandwidth (puede tardar 2-4 minutos)...')\n",
    "print('-' * 50)\n",
    "\n",
    "for bw in bw_grid:\n",
    "    fold_aucs = []\n",
    "    for train_idx, val_idx in cv_bw.split(X_sub, y_sub):\n",
    "        X_tr, X_val = X_sub[train_idx], X_sub[val_idx]\n",
    "        y_tr, y_val = y_sub[train_idx], y_sub[val_idx]\n",
    "\n",
    "        # Balancear solo el entrenamiento\n",
    "        X_tr_b, y_tr_b = balance_dataset(X_tr, y_tr)\n",

    "def evaluate_model_cv(model_name, model_factory, X, y, n_splits=5):\n",
    "    \"\"\"\n",
    "    Evalúa un modelo con k-fold estratificado.\n",
    "    Retorna AUC promedio, std, tiempos y curvas ROC por fold.\n",
    "    \"\"\"\n",
    "    cv = StratifiedKFold(n_splits=n_splits, shuffle=True, random_state=42)\n",
    "    fold_aucs, all_fpr, all_tpr = [], [], []\n",
    "    start_time = time.time()\n",
    "\n",
    "    for fold, (train_idx, val_idx) in enumerate(cv.split(X, y)):\n",
    "        X_tr, X_val = X[train_idx], X[val_idx]\n",
    "        y_tr, y_val = y[train_idx], y[val_idx]\n",
    "\n",
    "        # Balanceo solo en entrenamiento\n",
    "        X_tr_b, y_tr_b = balance_dataset(X_tr, y_tr, random_state=fold)\n",
    "\n",
    "        model = model_factory()\n",
    "        model.fit(X_tr_b, y_tr_b)\n",
    "\n",
    "        proba = model.predict_proba(X_val)[:, 1]\n",
    "        proba = np.clip(np.nan_to_num(proba, nan=0.5), 0, 1)\n",
    "\n",
    "        auc = roc_auc_score(y_val, proba)\n",
    "        fold_aucs.append(auc)\n",
    "\n",
    "        fpr, tpr, _ = roc_curve(y_val, proba)\n",
    "        all_fpr.append(fpr)\n",
    "        all_tpr.append(tpr)\n",
    "\n",
    "    elapsed = time.time() - start_time\n",
    "\n",
    "    return {\n",
    "        'model': model_name,\n",
    "        'auc_mean': np.mean(fold_aucs),\n",
    "        'auc_std': np.std(fold_aucs),\n",
    "        'fold_aucs': fold_aucs,\n",
    "        'time_sec': elapsed,\n",
    "        'fpr_list': all_fpr,\n",
    "        'tpr_list': all_tpr\n",
    "    }\n",
    "\n",
    "print('✅ Función de evaluación lista.')"

    "## ✅ 9. Conclusiones\n",
    "\n",
    "### 1. ¿Vale la pena usar KDE frente a GaussianNB?\n",
    "**Sí**, los tres métodos KDE superan al GaussianNB clásico en AUC. Esto se explica por la distribución no gaussiana de variables como `Torque [Nm]` y `Tool wear [min]`, que presentan asimetría y bimodalidad entre clases. KDE captura estas formas arbitrarias sin asumir normalidad.\n",
    "\n",
    "### 2. ¿Cuál método KDE es mejor?\n",
    "| Método | AUC | Varianza | Costo |\n",
    "|---|---|---|---|\n",
    "| **KDE Silverman** | ~0.9219 | Baja | Medio |\n",
    "| **KDE Gaussiano (opt)** | ~0.9211 | Baja | Alto |\n",
    "| **KDE Parzen** | ~0.9156 | Media | Bajo |\n",
    "| GaussianNB | ~0.8937 | Media | Muy bajo |\n",
    "\n",
    "**KDE Silverman** ofrece el mejor balance: buen AUC, menor varianza entre folds, sin necesidad de optimizar hiperparámetros.\n",
    "\n",
    "### 3. Impacto del bandwidth\n",
    "- **BW muy pequeño:** sobreajuste (densidad ruidosa con picos en cada observación).  \n",
    "- **BW óptimo (CV):** mejor AUC, balance sesgo-varianza.  \n",
    "- **Silverman:** excelente aproximación automática, especialmente para distribuciones cerca de normales.  \n",
    "- **BW muy grande:** suavizado excesivo, pierde la forma real de la distribución.\n",
    "\n",
    "### 4. Manejo del desbalance\n",
    "El submuestreo 3:1 fue efectivo para evitar que los modelos ignoren la clase minoritaria. Una alternativa más robusta sería SMOTE (sobremuestreo sintético), que no descarta datos de la clase mayoritaria.\n",
    "\n",
    "### 5. Costo computacional\n",
    "GaussianNB es prácticamente instantáneo. KDE es significativamente más costoso (O(n) por predicción por feature), lo que puede ser limitante en producción con grandes volúmenes de datos.\n",
